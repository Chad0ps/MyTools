usage: infoFiles.py [-h] [--file FILE] [--url URL] [--save]

Sensitive information file
```cmd
optional arguments:
  -h, --help            show this help message and exit
  --file FILE, -f FILE  url file
  --url URL, -u URL     may be you only want to test one url
  --save, -s            save scan infos
```
f指定url文件，u指定单一url,s,将扫描信息保存
